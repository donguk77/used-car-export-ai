## 📌 먼저 알아야 할 것 — 형식이 동일하지 않습니다

**공통 골격(80%)** + **국가별 부가 요건(20%)** 구조입니다.

### 공통으로 들어가는 것 (모든 국가 동일)

거의 모든 국가는 이 5가지를 영문으로 요구해요:

1. **Commercial Invoice** (상업송장)
2. **Packing List** (포장명세서) — VIN 표기 필수
3. **Bill of Lading (B/L)** — 선사 발급
4. **Certificate of Origin (C/O)** — 대한상공회의소 발급
5. **Export Declaration / 수출신고필증** — 한국 관세청 (관세사 EDI)

### 국가마다 다른 부가 요건

여기서부터 갈라집니다:

| 차이가 발생하는 영역 | 예시 |
|---------------------|------|
| **언어 요구사항** | 리비아·알제리는 아랍어 **번역 필수**. 도미니카·칠레는 스페인어 권장. 러시아권은 영문 OK |
| **선적 전 검사 (PSI)** | 케냐(JEVIC+QISJ), 탄자니아·우간다(JEVIC), 모잠비크(Intertek), 나이지리아(SONCAP) — 이건 의무이고, 검사 안 받으면 도착해도 통관 불가 |
| **영사관 인증 (Legalization)** | 리비아·도미니카·칠레·페루는 한국 주재 해당국 대사관/영사관에서 **C/O와 인보이스에 도장**을 받아야 함. 이거 빼먹으면 통관 거부 |
| **사전 등록 시스템** | 리비아 ACI, 알제리 ECTN, 이집트 ACI/Nafeza, 모잠비크 GUCE — 선적 **전에** 도착국 시스템에 데이터 미리 등록해야 함 |
| **HS Code별 절대세율** | 베트남은 16인승 이상 150% 등 차종별로 세율표가 따로 |
| **추가 서류** | Tunisia처럼 사실상 신차만 받는 국가 / 중앙아시아처럼 우회수출 조사 대비 EUC(End-User Certificate) 필요 |

그래서 "한 가지 서식으로 다 자동화"가 안 되고, 국가별로 룰 엔진이 갈라져야 해요.

## 2. 국가별 상세 — 각국이 어떻게 다른가

여기서부터는 표로 다 못 담는 디테일입니다. 각 국가에서 **AI 에이전트가 사람을 대신해서 처리해야 할 차이점** 위주로 정리.

### 1️⃣ 🇱🇾 리비아 — 한국 직수출 1위 (12.1만대)

**왜 까다로운가:** 동·서 정부 분열로 항구별 적용 규칙이 다름. 2024년부터 ACI(Advance Cargo Information) 사전 등록 의무화. 2024년부터 관세 산정 방식이 컨테이너 단위 → **대당(per unit) 산정**으로 바뀌어서 단가가 올라감.

**AI가 자동화해야 할 것:**
- **모든 서류 아랍어 번역** (인보이스, 패킹리스트, C/O까지). 영문만 제출하면 통관 거부
- **C/O 영사관 인증(Legalization)**: 대한상공회의소 발급 → 주한 리비아대사관에서 도장. 선적 **전에** 끝나야 함
- **ACI 사전 등록**: 컨테이너 적재 전에 customs.gov.ly에 등록. 등록 안 하면 출발항에서 적재 거부
- **CBL(Central Bank of Libya) 결제 채널 확인**: 송금이 승인 은행 채널 통해야 통관됨

**연식·차량 규칙:**
- 10년 이상 차량 수입 금지 (2022년 이후)
- 럭셔리카 수입 금지 (HS 분류로 사전 차단 필요)
- EV는 VAT·관세 면제 (Euro 4 이상 충족 시)

**주의:** "이스라엘 원산 물품 일체 금지". 부품에 이스라엘 원산이 섞이면 전체 차량 통관 거부. 차량 자체는 한국산이라 보통 문제없지만, 부품 교환 이력에 주의.

**사용 언어:** 모든 비즈니스 커뮤니케이션 **아랍어**. 일부 바이어는 영어/이탈리아어 가능. **AI 메일 기본값: 아랍어 + 영어 병기**.

### 2️⃣ 🇰🇬 키르기스스탄 — 한국 수출액 1위 (14.9억$) 🚨

**최대 위험:** 사실상 러시아 재수출 통로. **2025.10 부산경찰청 적발 사례 (40여 대 우회수출)**. 한국에서는 자유롭게 보내지지만, 수출자가 형사 처벌받는 사례가 이미 발생.

**AI가 반드시 해야 할 것:**
- **컴플라이언스 사전 차단 룰** (이게 가장 중요):
  - 2,000cc 초과 내연기관 → 산업통상자원부 상황허가 받아야 수출 가능
  - 하이브리드/EV → 배기량 무관, 전부 상황허가 대상
  - 5만 달러 이상 승용차 → 수출 금지 품목
  - 신규 바이어 + 위 조건 충족 → 자동 "HIGH RISK" 플래그
- **Yestrade 우려거래자 조회** 자동 실행
- **End-User Certificate (EUC)** 요청 — 바이어가 "최종 사용자가 키르기스스탄이고 재수출 안 한다"는 서약서 제출

**연식·차량 규칙:**
- 자체 연식 제한은 사실상 없음
- EAEU(유라시아경제연합) 회원국이라 카자흐스탄·러시아·아르메니아·벨라루스와 통합 통관

**사용 언어:** 키르기스어가 공식이지만 **비즈니스는 거의 100% 러시아어**. AI 메일 기본값: **러시아어**. 단 우회수출 의심 메시지/조건이 들어가면 AI가 자동 거부해야 함.

**참고:** 카자흐스탄·타지키스탄·아르메니아도 동일 위험 카테고리.

### 3️⃣ 🇸🇾 시리아 — 2025년 신규 폭증 시장

**상황:** 2024.12 아사드 정권 붕괴 → 2025.3월 직수출 2만대, 4월 1.6만대. 1년 전엔 통계상 거의 없던 국가.

**AI가 주의해야 할 것:**
- 제재 변동성: 미국 OFAC, EU 제재가 2025-2026년에도 부분 완화/해제 진행 중. **선적 전 OFAC SDN 자동 조회 필수**
- 결제 인프라 미비: 송금 채널 제한적. T/T 직접보다 UAE·요르단 거점 결제가 많음
- 통관 절차 격변 중: 과도정부 체제라 항구별 운영 방식이 매월 바뀜. **고정 룰을 박지 말고 매뉴얼 + 알림 모드**로 운영

**연식·차량 규칙:**
- 정권 교체 후 사실상 모든 연식 수용 (단 변동 가능)
- 재건 수요라 SUV·1톤 트럭·중형 세단 강세

**사용 언어:** 아랍어. 단 시리아 방언(Levantine Arabic)이라 표준 아랍어와 미세하게 다름. 격식 비즈니스 문서는 **표준 아랍어(MSA)** OK.

**주의:** 요르단·UAE 경유 재수출이 더 큰 비중. 직수출 외에 **Notify Party를 시리아로 두고 Consignee를 요르단/UAE로 거는 거래 패턴**이 일반적. AI는 이 구조를 인식해야 함.

### 4️⃣ 🇯🇴 요르단 — 안정적 거점 (양 4위)

**왜 좋은가:** 정세 안정 + 영어 비즈니스 + 시리아 재수출 통로 + 친환경차 수출 1위(2024년 3,321대).

**AI가 자동화할 것:**
- **ACID(Advance Customs Import Declaration)** 사전 등록 — 출발 48시간 전
- 친환경차(HEV/EV) 인보이스에 명시 — 관세 우대 받을 수 있음
- 시리아 재수출 의심 패턴 모니터링 (큰 볼륨이 짧은 기간 + 신규 바이어)

**연식·차량 규칙:**
- 5년 (정책 변동 잦음, 2024-2025 검토 중)
- 연식보다 **배기량 기반 누진세** 영향 큼. 1500cc 이하가 압도적 인기

**사용 언어:** 아랍어가 공식이지만 **영어 비즈니스 충분**. 메일은 **영어 기본**, 격식 문서만 아랍어 병기.

**항구:** Aqaba 단일항이라 단순.

### 5️⃣ 🇪🇬 이집트 — 사실상 신차 위주 시장 (양 5위)

**진입장벽:** 이집트는 중고차 일반 수입이 사실상 어려움. 등장하는 한국 중고차는 대부분 **개인 이주차** 또는 **외교관/특수 분류**. 일반 비즈니스 채널은 신차 중심.

**그럼에도 불구하고 양 5위인 이유:** 휘발유차 + 인기 모델(쏘나타 등) 한정으로 일부 채널 존재. 단 정책이 매우 보수적.

**AI가 처리해야 할 것:**
- **Nafeza (ACI)** 사전 등록 — 모든 수입 화물 의무
- **GOEIC 인증** — 이집트 수출입통제기구 사전 검사
- **인보이스 아랍어 필수** + 이집트상공회의소 인증

**사용 언어:** 아랍어. 비즈니스는 영어 가능하지만 **모든 공식 서류 아랍어 번역 필수**.

**전략 권고:** 이집트는 PoC 대상에서 **제외 권장**. 진입장벽 대비 보상이 낮음.

**상황:** 2024년 한국 신차 수출 8,029대 (+6.5%), 중고차는 +248%. EAEU 회원국이라 러시아와 통합. **카자흐스탄도 키르기스스탄과 같은 우회수출 위험 카테고리**.

## 4. 핵심 정리 — 한 페이지 요약

**공통:** 인보이스/패킹리스트/B/L/C/O/수출신고필증 5종은 모든 국가 동일.

**갈라지는 지점은 4가지:**
1. **언어** — 영어(동·서아프리카·UAE·요르단) / 아랍어(북아프리카·중동) / 스페인어(중남미) / 러시아어(중앙아시아) / 프랑스어(알제리·서아프리카 일부)
2. **선적 전 검사 (PSI)** — 케냐·탄자니아·우간다(JEVIC) / 나이지리아(SONCAP) / 모잠비크(Intertek) / 알제리(COC) / 멕시코(NOM)
3. **영사관 인증 (Legalization)** — 리비아·도미니카·칠레·페루·볼리비아·에콰도르 — 한국 주재 해당국 영사관 도장
4. **사전 등록 시스템** — 리비아 ACI / 이집트 Nafeza / 알제리 ECTN / 케냐 iCMS / 나이지리아 B'Odogwu / 가나 ICUMS / 탄자니아 TANCIS / 칠레 SNA / 멕시코 VUCEM / 도미니카 DGA

**AI가 가장 큰 가치를 만들 수 있는 곳:**
- 국가 입력 → 4가지 갈래(언어/PSI/인증/사전등록) 자동 분기
- 차량 정보 입력 → 해당 국가 통관 가능 여부 즉시 판정
- 영사관 인증·사전등록 등 사람이 까먹기 쉬운 항목 자동 알림
- **컴플라이언스 (러시아 우회 + OFAC)** 자동 사전 차단
