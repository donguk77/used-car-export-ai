- **16개 언어 지원**, 100여 개국 바이어 네트워크

- KB차차차와 제휴 (간접 수출 모델)

→ **"한국에서 처음 만드는 중고차 수출 플랫폼"이 아닙니다.** 우리 캡스톤은 **"이미 존재하는 마켓플레이스를 AI 에이전트로 어떻게 뛰어넘을 것인가"** 의 문제로 재정의해야 함.

### 🎯 핵심 차별화 지점은 BeForward·오토위니의 **결정적 약점**에 있음

Trustpilot 리뷰 분석 결과 BeForward의 일관된 불만 사항:
1. **"결제 전엔 응답 잘 하다가 결제 후 사라진다"** (사후관리 공백)
2. **WhatsApp 응답 부재** (영업시간/시차 한계)
3. **클레임 처리 지연·회피** (분쟁 해결 시스템 부재)
4. **선적 후 사고·운송 사고 책임 회피** (transit 단계 가시성 부족)

오토위니도 사실상 같은 문제 — **셀러 등록 + 바이어 매칭까지만 자동화**, 그 이후는 사람이 다 처리해야 함.

→ **AI 에이전트의 진짜 가치는 "Pre-sale + Sale + Post-sale 전 과정 24/7 자동 응대"**. 이게 우리가 잡아야 할 포지션.

## 1. 경쟁사 4개 카테고리 분석

이 카테고리가 우리 앱이 가장 직접 비교될 곳. 사용자가 "한국판 BeForward 만든다"고 인식할 가능성 큼.

#### 공통 핵심 기능 (모두 갖고 있음)
- 차량 검색·필터 (브랜드/모델/연식/주행/색/가격)
- 30~50장 사진 + 동영상
- VIN/사고이력
- 옥션 등급/검사시트 PDF
- WhatsApp/Viber 영업
- 다국어 사이트 (UI 번역)
- 견적·예약·결제 흐름

#### 그들이 안 하는 것 (= 우리의 빈틈)
- **사후관리 자동화** (선적 후 → 도착 → 통관 → 등록까지 알림)
- **클레임/분쟁 자동 트리아지** (어떤 케이스가 사진/리포트 부족인지 자동 판별)
- **다국어 메일 자동 작성** (영업 채팅 외 격식 메일은 사람 작성)
- **국가별 통관 사전 검증** (이 차가 그 나라에서 통관되는지 자동 판단)
- **컴플라이언스 사전 차단** (러시아 우회·OFAC 자동 검사)

#### 한국 측 경쟁 환경 시사점

오토위니가 "마켓플레이스" 포지션을 이미 잡고 있어서 우리가 **같은 형태로는 못 이김**. 대신 **"AI 에이전트가 영세 수출업체의 영업·서류·번역·컴플라이언스를 다 대행"** 하는 SaaS/도구 포지션이 빈 공간.

영세 업체 1,000여 곳 (인천 송도 중심)이 잠재 고객. 이들은 오토위니에 매물 올리고 있지만 **메일 응답·서류·번역은 다 본인이 처리**. 여기를 AI가 대신.

## 2. 차별화 포지셔닝

### 2.1 우리가 가야 할 포지션

```
              [마켓플레이스]
               (BeForward·SBT·오토위니)
                       ↑
                       │
                       │
[수출업체 도구] ─────●───── [B2B 영업 AI]
   (DealerCenter)       (Swirl·Impel·Numa)
                       │
                       │
                       ↓
            [한국 영세 수출업체 +
             도착국 컴플라이언스 +
             다국어 자동 응대 + 서류 자동화]
                  ★ 우리 ★
```

### 2.2 한 줄 포지셔닝

> **"한국 영세 중고차 수출업체를 위한, 도착국 통관 룰·다국어 응대·수출 서류·컴플라이언스를 24/7 자동화하는 AI 에이전트 SaaS"**

마켓플레이스(오토위니·BeForward)가 아니라 **그 위에서 동작하는 자동화 도구** 포지션.

오토위니에 매물 올리는 영세업체가 **문의 응대·견적·서류·통관 검증·번역까지 본인이 다 하는 부담**을 AI가 대신 처리.

### 2.3 차별화 5가지 핵심 (오토위니/BeForward가 안 하는 것)

| # | 우리가 하는 것 | 경쟁사 현상태 | 우리 차별화 |
|---|--------------|--------------|------------|
| 1 | **국가별 통관 사전 검증** | 사람이 찾아봄 | "이 차는 케냐 통관 가능, 도미니카 불가" 자동 판정 |
| 2 | **다국어 영업 메일 자동 작성** | 영어만, 채팅만 | 영/아랍/스페인/러/불 5개 언어 격식 메일 자동 |
| 3 | **수출 서류 자동 생성** | 사람이 작성 | 인보이스/패킹리스트/SI/C/O 자동, VIN/스펙 자동 채움 |
| 4 | **컴플라이언스 자동 차단** | 거의 없음 | 러시아 우회/OFAC/Yestrade 자동 조회 |
| 5 | **선적 후 24/7 추적·알림** | 사람 의존 | B/L → 도착항 → 통관 → 등록까지 자동 알림 (다국어) |

## 4. MVP 핵심 기능 (10개 이내)

캡스톤 발표/시연용 최소 기능. 우진님 시간 한정 + 발표 임팩트 균형.

### 🎯 MVP 10개 기능

| # | 기능 | 설명 | 우선순위 | 발표 임팩트 |
|---|------|------|----------|-----------|
| 1 | **차량 정보 입력 + VIN 자동 디코딩** | NHTSA vPIC API → make/model/year/engine 자동 채움 | ★★★ | 첫 5초에 보여주기 좋음 |
| 2 | **국가별 통관 가능성 즉시 판정** | 차량 입력 → "케냐 ✅ 통관, 도미니카 ❌ (10년 초과)" | ★★★ | 가장 직관적 차별화 |

| 4 | **다국어 견적 메일 자동 생성** | 영/아랍/스페인 3개 언어, 5종 시나리오 | ★★★ | LLM 핵심 기능 시연 |
| 5 | **인보이스 자동 생성 (PDF)** | 차량+바이어 정보 입력 → PDF 다운로드 | ★★★ | 즉시 사용 가능한 결과물 |

| 7 | **다국어 대시보드 (한국어/영어)** | 셀러용 UI | ★★ | 기본기 |

## 6. 캡스톤 발표 차별화 후크 3가지

발표할 때 다음 3가지를 강조하면 단순 마켓플레이스 클론으로 안 보임:

### 후크 1: "시리아 재건 마켓 — 2025년 게임체인저"
- 2025.3월 직수출 2만대 (1년 전엔 거의 0)
- 우리 시스템은 시의성 있는 신규 시장 자동 추적
- 발표 슬라이드: 1년 전 그래프 vs 2025년 폭증 그래프

### 후크 2: "AI가 형사처벌을 막는다"
- 2025.10 부산경찰청, 키르기스스탄 우회수출 적발 (40대, 구속)
- 우리 AI가 자동 차단 시연
- 발표 슬라이드: 실제 뉴스 헤드라인 + AI BLOCKED 메시지

### 후크 3: "한국 영세업체 1,000여 곳 vs 1명의 AI"
- 인천 송도 영세 수출업체 1,000여 곳, 대부분 1~2인 운영
- 영어/아랍어/스페인어/러시아어 동시 처리 불가능
- 우리 AI는 5개 언어 × 24/7 × 통관 룰 + 컴플라이언스 동시 처리
- 발표 슬라이드: 사람 1명 → AI 1개 비교 도식
