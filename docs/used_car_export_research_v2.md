## 0. TL;DR (v2)

세 가지 핵심 발견:

1. **타겟 시장은 "동남아·아프리카·남미"가 아니라 "중동·중앙아시아·아프리카·일부 남미"**. 동남아는 한국 직수출 거의 없음 (보호장벽). 남미는 도미니카공화국(8위)·칠레(9위)·코스타리카·아제르바이잔(중간 경유) 정도가 실질 시장.
2. **2025년 신규 폭발 시장 = 시리아.** 아사드 정권 붕괴(2024.12) 후 재건 수요 폭증. 2025년 3월 직수출 2만대, 4월 1.6만대. 요르단·UAE 경유까지 합치면 더 큼. 캡스톤 지향에 시의성 있음.
3. **러시아 우회수출 적발 사례 발생** — 2025년 10월 부산경찰청, 키르기스스탄 국적자 2명 구속. 우리 에이전트는 **컴플라이언스 모듈 필수**. 안 그러면 "AI가 우회수출 도와줬다"는 헤드라인 위험.

## 1. 시장 컨텍스트 — 한국 중고차 수출 실제 지도 (v2 재작성)

### 1.1 매크로 데이터 (검증 통과)

| 항목 | 수치 | 출처 |
|------|------|------|
| 2024년 한국 중고 승용차 수출량 | 53.3만대 | 산업연구원 산업분석 (2025.05) |
| 2024년 한국 중고 승용차 수출액 | 47.4억 달러 | 동상 |
| 2024년 중고차(전체, 승용+상용) | **62.6만대** | 한국무역협회 |
| 2025년 1~4월 누적 중고차 수출 | **29.7만대 / 39.2억 달러** | 한국중고차유통연구소 |
| 2025년 4월 단월 수출 | **8만대 / 7.6억 달러 (역대 최고)** | 동상 |
| 2025년 10월 누적 중고차 수출 증가율 | **전년 대비 +78%** | 한국자동차산업협동조합 |
| 자동차 수출 중 중고차 비중 | 7.2% (2024) → **12.3% (2025)** | 동상 |
| 인천항 처리 비중 | 약 **80%** | 인천항만공사 (IPA) |
| 글로벌 시장 점유율 | **약 8% (4위)** | EU 49% > 일본 26% > 미국 18% > 한국 8% |

### 1.2 한국 중고차 수출 상위국 (2024 실수치)

| 순위 | 국가 | 수출량 (대) | 비고 |
|------|------|-------------|------|
| **1위** | 🇱🇾 **리비아** | 121,000 | 북아프리카 재수출 거점 (튀니지·알제리·수단으로 재수출). 저가 위주 |
| **2위** | 🇰🇬 **키르기스스탄** | 72,000 | 사실상 **러시아 재수출 통로** (대외무역법 제재 대상) |
| 3위 | 🇰🇿 카자흐스탄 | (미공개) | 키르기스스탄과 유사 패턴 |
| 4위~ | 🇯🇴 요르단, 🇪🇬 이집트, 🇹🇷 튀르키예, 🇦🇪 UAE | | 중동권. 시리아 재수출 |
| ~10위 | 🇩🇴 **도미니카공화국**, 🇨🇱 **칠레**, 🇦🇿 아제르바이잔 | | 남미·코카서스 진입 시장 |

> 📊 **수출액 기준 1위는 키르기스스탄 (14.9억 달러, 31.4%)** — 중앙아시아는 양보다 단가가 높음 (러시아행 고급 차량). 리비아는 양 1위지만 저가라 금액 비중은 낮음.

### 1.3 2025년 신규 폭발 시장: 🇸🇾 시리아

- 2024년 12월 아사드 정권 붕괴 → 과도정부 체제에서 중고차 수입 폭증
- **2025년 3월 시리아 직수출: 20,183대 / 4월: 16,063대**
- 요르단·UAE 경유 재수출 물량까지 합치면 더 큼 (정확한 통계 없음)
- 1년 전만 해도 시리아는 한국 수출 통계에 거의 없던 국가
- **시사점:** 캡스톤 발표 시점(2026)에도 여전히 핫. "2025-2026 시리아 재건 수요" 자체가 좋은 발표 후크

### 1.4 동남아·남미·아프리카 — 실제 시장 가능성 재평가

| 지역 | v1 평가 | v2 정정 |
|------|--------|---------|
| **동남아** (베트남·인니·태국·말련·필리핀) | "점진적 자유화 시장" | **❌ 한국 직수출 거의 불가능.** 자국 산업 보호 강함. 5~10년 제한 + 1.5~150% 관세. 캄보디아·미얀마는 일본 RHD 압도적 |
| **아프리카** | "주력 시장" | **✅ 맞음.** 단 한국 비중은 **리비아·이집트** 외에는 **케냐·우간다·탄자니아 = 일본 80%+ 점유**. 한국이 새로 뚫을 가능성 있는 곳은 **나이지리아·앙골라·DR콩고** |
| **남미** | (언급 없음) | **✅ 7~9위권 시장 존재.** 도미니카공화국·칠레는 한국 중고차 상위 10위권 단골. 코스타리카·아제르바이잔도 SUV·디젤 강세 |
| **중앙아시아** | (언급 약함) | **🚨 양면성.** 키르기스스탄·카자흐스탄·타지키스탄 = **수출액 기준 최대 시장 + 러시아 우회 리스크 최대**. 컴플라이언스 신경 안 쓰면 형사처벌 |
| **중동** | (언급 약함) | **✅ 핵심.** 요르단·UAE·이집트 = 시리아 재수출 통로. 사우디는 진입장벽 있음 |

> **AI 에이전트 1차 PoC 권장:** 리비아·도미니카공화국·칠레 중 1개. 리비아는 시장 큰데 정세 불안. **도미니카공화국이 PoC 최적** (10위권 안정 시장 + LHD + 영어/스페인어로 다국어 검증 가능 + 정세 안정 + 통관 절차 표준화).

## 2. 차량 정보 (Vehicle Data) 필수 필드

(v1과 동일, 검증 결과 변경 사항 없음. 일부 표현 명확화)

### 2.1 식별 정보 (Identification)

| 필드 | 한글명 | 필수 | 비고 |
|------|--------|------|------|
| `vin` | 차대번호 (VIN) | ★★★ | 17자리. 한국차는 "K"로 시작. 도난·말소 여부 검증 키. NHTSA vPIC API로 디코딩 가능 |
| `vehicle_id` | 내부 매물 ID | ★★★ | 자체 발번 |
| `registration_no` | 차량등록번호 | ★★ | 말소등록 시 사용 |
| `registration_date` | 최초등록일 | ★★★ | **케냐·도미니카공화국 등은 최초등록일 기준 연식 제한** |
| `manufacture_date` | 제조연월 | ★★★ | 일부 국가는 제조연도 기준 (나이지리아 등) |

### 2.2 차량 기본 사양

| 필드 | 한글명 | 비고 |
|------|--------|------|
| `make` | 제조사 | Hyundai, Kia, Genesis, KGM(KG모빌리티), Renault Korea 등 |
| `model` | 모델명 | 검증 결과 **2025년 1~4월 한국 중고차 수출 1위는 셰보레 트랙스, 2위 코나, 3위 아반떼**. SUV·소형 강세 |
| `trim` / `grade` | 트림/등급 | "2.0 Premium" 같이 옵션 패키지명 |
| `year` | 연식 (모델이어) | 검색 필터의 1차 축 |
| `body_type` | 차종 | Sedan, SUV, Pickup, Van, Bus, Truck. **신흥국은 SUV·1톤 트럭·픽업이 주력** (러우전쟁 후 트렌드) |
| `fuel_type` | 연료 | Gasoline, Diesel, LPG, Hybrid, EV. **러시아: 2,000cc 초과 내연기관 + 하이브리드/EV 전체 상황허가 대상** (2024.2.24 개정) |
| `engine_displacement` | 배기량 (cc) | 러시아 우회수출 핵심 변수. 5만 달러 이상 승용차도 별도 수출금지 품목 |
| `transmission` | 변속기 | A/T, M/T |
| `drivetrain` | 구동방식 | FWD, RWD, AWD, 4WD |
| `steering` | **핸들 위치 (LHD/RHD)** | **국가별 룰 엔진의 1차 차단 변수.** §5 매트릭스 참고 |
| `seats` | 좌석 수 | HS코드 분류 영향 (8702: 10인 이상, 8703: 9인 이하) |
| `color_exterior` / `color_interior` | 외장/내장색 | **더운 국가는 흰색 선호** (리비아·시리아·사우디). 검정 대비 100~150만원 차이 |
| `mileage_km` | 주행거리 | UN/UNEP 기준: 5,000km 초과 = 중고 |
| `vin_decoded` | VIN 디코딩 결과 | NHTSA vPIC API 활용 |

```
   ↓
② 자동차 등록 말소
   - 자동차관리법 제13조
   - 시·도지사(차량등록사업소)에 신청
   - 자동차등록증, 등록번호판, 봉인 반납

   ↓
③ 상품화 (수리/세차/광택/판금)
   ↓
④ [컨테이너 운송 시만] 보세구역 반입 ★ v2 정정
   - 관세법 제243조 + 시행령 제248조의2
   - 「수출통관 사무처리에 관한 고시」 제7조의3
   - 2017년 4월부터 시행
   - 공항만 지역 보세창고 등에 한정
   - ⚠️ RoRo·BULK 운송은 보세구역 반입 면제
   ↓
⑤ 세관 수출신고 (관세사 EDI)
   - 컨테이너: 신고 구분 [B] 코드
   - 필수서류: Invoice, Packing List(차대번호 표기),
              자동차말소등록사실증명서,
              [컨테이너] 차량적재 사진 + 컨테이너번호 사진,
              쇼링 전후 전자이미지, 보세구역 반입계
   ↓
⑥ 통관 심사 (도난·말소·우회수출 여부 전산 검증)
   ↓

   ↓

```

### 4.3 컴플라이언스 자동 검증 룰 (제안)

```python
def compliance_check(buyer, vehicle):
    flags = []
    
    # 1. 러시아 직수출 차단
    if buyer.country == "RU" or buyer.country == "BY":
        flags.append("CRITICAL: Direct export to RU/BY blocked")
    
    # 2. 우회수출 의심
    if buyer.country in ["KG", "KZ", "TJ", "AM", "UZ"]:
        if vehicle.engine_cc > 2000 or vehicle.fuel == "EV" or "HEV" in vehicle.fuel:
            flags.append("HIGH: Strategic items (>2000cc, EV/HEV) to RU-adjacent country - 상황허가 필요")
        if vehicle.list_price_usd > 50000:
            flags.append("HIGH: USD 50k+ vehicle to RU-adjacent country - 수출금지 검토")
    
    # 3. Yestrade 우려거래자
    if yestrade_query(buyer.tax_id) == "concerned":
        flags.append("CRITICAL: Yestrade 우려거래자")
    
    # 4. OFAC SDN
    if ofac_query(buyer.company_name) == "match":
        flags.append("CRITICAL: OFAC SDN list match")
    
    return flags
```

이 모듈만 잘 만들어도 캡스톤 발표에서 "단순 마켓플레이스가 아니라 컴플라이언스 에이전트"로 차별화됨.

### 5.6 룰 엔진 핵심 규칙 6가지 (v2 보강)

1. **연식 계산 기준 분기** — "최초등록일" / "제조연도" / "선적일" 별도 메타데이터
